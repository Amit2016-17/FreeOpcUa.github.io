Freeopcua.github.io

Free OPC-UA is C++ library develop server and client OPC-UA applications. Python bindings can be found in the opcua-python repository. The library is under development but works will all tested clients. It support addressspace browse, read, write and address space modification as well as subscriptions and datachange events.

history is not yet supported

see examples in exe folder or the examples in python repository
